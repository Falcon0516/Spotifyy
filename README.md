## Features

The application includes the following key screens and functionalities:

* **Home Page**:
    * Displays a "Liked Songs" section with recently played or liked tracks[cite: 2].
    * Features a "Jump back in" section for recently played albums or playlists[cite: 3].
    * Highlights "New Music Friday" for handpicked new international music[cite: 4].
    * Presents "Friday's freshest playlists" with curated content[cite: 4].
    * Allows toggling between "All," "Music," and "Podcasts" views[cite: 4].
* **Search Page**:
    * Provides a search bar for content discovery[cite: 5].
    * Offers categories like "Music," "Podcasts," "Live Events," and "Home of I-Pop" to "Start Browse"[cite: 6].
    * Suggests "Discover something new" with hashtag-based categories (e.g., #pop edm, #krushfunk, #odecore)[cite: 7].
    * Includes "Browse all" sections such as "Made For You" and "New Releases"[cite: 8].
* **Your Library Page**:
    * Organizes content into "Playlists," "Podcasts," "Artists," and "Downloads" categories[cite: 9].
    * Features a "Recents" sorting option[cite: 9].
    * Lists saved items like "Liked Songs," "Your Episodes," and artists (e.g., "The Chainsmokers," "Linkin Park," "Marshmello")[cite: 10].
    * Displays "Local Files" and user-created playlists[cite: 11].
* **Premium Page**:
    * Promotes a premium subscription with a trial offer (e.g., "Try 2 months of Premium Individual for â‚¹119")[cite: 11].
    * Highlights benefits like ad-free music, offline listening, and playing songs in any order[cite: 12].
    * Mentions features like creating playlists and collaborative playlists[cite: 13].
* **Podcasts View (within Home Page)**:
    * Informs the user if no podcasts are followed yet[cite: 14].
    * Provides a "Browse podcasts" button for discovery[cite: 14].
